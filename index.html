About

The Void Universe is a website where you can read, watch and possibly help us. The Void Universe is written without libraries in pure code and is largely a case of Stack Overflow code. The creator is the user Yupiterhuer, whose real name is Aidar. The site is written mainly in the HTML programming language, and the functions of how to log in, write an article are not implemented in the site itself. The main purpose of the site is to open the environment of a small team that develop a little information and also create an office of professional programmers, IT specialists and various SEO.
